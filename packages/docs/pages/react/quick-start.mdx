# React Quick Start

## Define a typed wizard

Use `createReactWizardFactory` to define steps with components.

```tsx
import { createReactWizardFactory } from '@wizard/react';

type Context = { total: number };

const factory = createReactWizardFactory<Context>();
const { defineSteps, step } = factory;

function AccountStep() {
  return <div>Account</div>;
}

const steps = defineSteps({
  account: step({
    data: { email: '' },
    component: AccountStep,
    next: ['review'],
  }),
  review: step({
    data: { agreed: false },
    next: [],
  }),
});

export const wizard = factory.createWizard(steps, {
  context: { total: 0 },
});
```

## Bind wizard state in components

Use hooks with a wizard instance:

```tsx
import { useWizard, useWizardStep } from '@wizard/react';

function Screen() {
  const { step } = useWizard(wizard);
  const account = useWizardStep(wizard, 'account');
  return <div>{step} / {account.status}</div>;
}
```

## Optional provider

`WizardProvider` is optional and can be used to avoid passing the wizard argument in every hook call.
