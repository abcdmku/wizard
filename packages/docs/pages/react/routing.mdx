# Routing

`@wizard/react` exposes hook-based router integration.

## Generic router sync

```tsx
import { useSyncWizardWithRouter } from '@wizard/react';

useSyncWizardWithRouter(wizard, {
  param: 'step',
  toStep: (param) => (param === 'account' || param === 'review' ? param : null),
  toUrl: (step) => ({ to: `/checkout/${step}` }),
  navigate: ({ to }) => routerNavigate(to),
  getParam: () => currentRouteParams.step,
});
```

## TanStack Router sync

```tsx
import { useTanStackWizardRouter } from '@wizard/react';

useTanStackWizardRouter(wizard, {
  basePath: '/checkout',
  paramName: 'step',
  toStep: (param) => (param === 'account' || param === 'review' ? param : null),
});
```

## Notes

- Integration is side-effect free: it does not mutate wizard methods.
- Invalid route params should return `null` from `toStep`.
