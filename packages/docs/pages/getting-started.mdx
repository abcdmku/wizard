# Getting Started

This guide uses the current `@wizard/core` API.

## 1. Create a typed factory

```ts
import { createWizardFactory } from '@wizard/core';

type Context = {
  userId?: string;
  total: number;
};

const factory = createWizardFactory<Context>();
const { defineSteps, step } = factory;
```

## 2. Define steps

```ts
const steps = defineSteps({
  account: step({
    data: { email: '' },
    next: ['review'],
    validate: ({ data }) => {
      if (!data.email.includes('@')) throw new Error('Invalid email');
    },
  }),
  review: step({
    data: { agreed: false },
    next: [],
  }),
});
```

## 3. Create wizard instance

```ts
export const wizard = factory.createWizard(steps, {
  context: { total: 0 },
});
```

## 4. Navigate

```ts
wizard.setStepData('account', { email: 'user@example.com' });
await wizard.next();
console.log(wizard.step); // "review"
```

## Notes

- Current state is exposed through getters like `wizard.step`, `wizard.context`, and `wizard.data`.
- Use `wizard.helpers` for progress/status/navigation checks.
- Step wrappers are available through `wizard.getStep(name)` and `wizard.getCurrentStep()`.
