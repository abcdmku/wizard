# Getting Started

Wizard is a headless, type-safe state manager for multi-step flows. It works in any environment (Node, React, Vue, etc.).

## Installation

```bash
npm install @wizard/core
```

If you are using React, also install the adapter:

```bash
npm install @wizard/react
```

## Core Concept

At its heart, Wizard is just a state machine.

1.  **Define Steps**: What steps exist and how they connect.
2.  **Create Wizard**: Instantiate the machine.
3.  **Interact**: Call `next()`, `back()`, `setData()`.

## Basic Usage (Vanilla JS/TS)

```typescript
import { createWizard, step, defineSteps } from '@wizard/core';

// 1. Define Steps
const steps = defineSteps({
  intro: step({
    next: ['details']
  }),
  details: step({
    data: { name: '' },
    next: ['finish']
  }),
  finish: step({
    next: []
  })
});

// 2. Create Instance
const wizard = createWizard(steps);

// 3. Interact
console.log(wizard.state.step); // 'intro'

await wizard.next();
console.log(wizard.state.step); // 'details'

wizard.setStepData('details', { name: 'Alice' });
await wizard.next();
console.log(wizard.state.step); // 'finish'
```
