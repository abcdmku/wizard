## Running Examples

Each example is a standalone application. To run an example:

```bash
# Clone the repository
git clone https://github.com/user/wizard.git

# Navigate to example
cd wizard/examples/basic-form-wizard

# Install dependencies
pnpm install

# Start development server
pnpm dev
```

## Basic Form Wizard

A registration form demonstrating:
- Step validation
- Progress indicator
- Form state persistence
- Error handling

## Router Integration 

Shows how to:
- Sync wizard state with URL
- Use browser back/forward navigation
- Implement route guards
- Deep link to specific steps

## Dynamic Branching

Demonstrates:
- Conditional step navigation
- Context-based branching
- Dynamic validation rules
- Complex state dependencies

## With Persistence

Features:
- Auto-save on every change
- Session recovery
- Expiration handling
- Secure state storage
