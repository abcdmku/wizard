# Status System

## StepStatus values

`StepStatus` values:

- `unavailable`
- `optional`
- `current`
- `completed`
- `required`
- `skipped`
- `error`
- `terminated`
- `loading`

## Read status with helpers

Read status with helpers:

```ts
wizard.helpers.stepStatus('account');
wizard.helpers.availableStepNames();
wizard.helpers.completedStepNames();
```

## Set runtime status explicitly

Set runtime status explicitly:

```ts
wizard.markLoading('account');
wizard.markError('account', new Error('Validation failed'));
wizard.markSkipped('profile');
wizard.markTerminated('payment');
wizard.markIdle('account');
```
