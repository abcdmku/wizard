# `@wizard/react` API

Source of truth: `packages/react/src/index.ts`

## Canonical Usage

```tsx
import { WizardProvider, useWizard, useWizardStep } from '@wizard/react';
import { wizard } from './wizard';

function AccountStep() {
  const step = useWizardStep(wizard, 'account');
  return (
    <button
      onClick={() => {
        step.updateData({ email: 'user@example.com' });
        void step.next();
      }}
    >
      Continue
    </button>
  );
}

export function App() {
  const { step } = useWizard(wizard);

  return (
    <WizardProvider wizard={wizard}>
      <div>Current step: {step}</div>
      <AccountStep />
    </WizardProvider>
  );
}
```

Provider usage is optional. Hooks support either explicit wizard injection or provider-based lookup.
