## Sync export tables

Sync the MDX export tables:

```bash
pnpm --filter @wizard/docs run sync:api-ref
```

## Validate parity

Validate parity (CI-safe):

```bash
pnpm --filter @wizard/docs run check:api-parity
```

## Generate symbol docs

For generated symbol-level docs, run:

```bash
pnpm --filter @wizard/docs build:api
```
