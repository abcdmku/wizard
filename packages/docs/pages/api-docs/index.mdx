## TypeScript Support

All packages are written in TypeScript and provide full type definitions. The wizard system is deeply type-safe, catching errors at compile time for:

- Invalid step names
- Incorrect data shapes
- Context mutations
- Navigation paths
- Helper method usage

## Import Examples

```typescript
// Core library
import { createWizard, type WizardConfig } from '@wizard/core';
import { selectors } from '@wizard/core';

// React adapter
import {
  WizardProvider,
  useWizard,
  useWizardState,
  useWizardActions
} from '@wizard/react';

// Router integration
import {
  useSyncWizardWithRouter,
  useTanStackRouterSync
} from '@wizard/react';
```
