# State Management

Wizard state is available through properties and store access.

## Core state access

```ts
wizard.step;     // current step id
wizard.context;  // readonly context
wizard.data;     // partial step data map
wizard.errors;   // step error map
wizard.history;  // navigation history
```

## State updates

```ts
wizard.updateContext((ctx) => {
  ctx.userId = 'user-123';
});

wizard.setStepData('account', { email: 'user@example.com' });
wizard.updateStepData('account', (current) => ({
  ...current,
  email: 'next@example.com',
}));
```

## Meta and errors

```ts
wizard.setStepMeta('account', { label: 'Account' });
wizard.updateStepMeta('account', { description: 'Enter your email' });

wizard.markError('account', new Error('Invalid email'));
wizard.clearStepError('account');
```

## Snapshot

For full low-level state access use:

```ts
const state = wizard.store.state;
```

`wizard.helpers.snapshot()` returns the same state shape for convenience.
