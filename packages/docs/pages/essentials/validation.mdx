# Validation

Use `validate` on each step.

```ts
const factory = createWizardFactory<{}, Error>();
const { defineSteps, step } = factory;

const steps = defineSteps({
  account: step({
    data: { email: '' },
    next: ['review'],
    validate: ({ data }) => {
      if (!data.email.includes('@')) {
        throw new Error('Invalid email');
      }
    },
  }),
  review: step({
    data: { agreed: false },
    next: [],
  }),
});
```

Validation runs on `wizard.next()` before transition.

## React error handling

```tsx
import { useStepError } from '@wizard/react';

const error = useStepError(wizard, 'account');
error?.message;
```

Or with a step wrapper:

```tsx
const step = useWizardStep(wizard, 'account');
step.error; // same underlying error
```
