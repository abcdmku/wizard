# Node Saga Wizard

Path: `examples/node-saga-wizard`

Highlights:

- `createWizardFactory` usage in a non-React environment
- CLI-driven multi-step order flow
- typed handlers for each step payload

Run:

```bash
pnpm --filter node-saga-wizard-example start
pnpm --filter node-saga-wizard-example start:auto
pnpm --filter node-saga-wizard-example build
```
