# Basic Form Wizard

## Path

Path: `examples/basic-form-wizard`

## Highlights

Highlights:

- `createReactWizardFactory` pattern
- per-step `validate` with typed `data`
- `useWizard` + `useWizardStep` hooks
- component-per-step rendering via step `component`

## Run

Run:

```bash
pnpm --filter basic-form-wizard dev
pnpm --filter basic-form-wizard build
```
