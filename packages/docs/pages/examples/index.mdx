## Running Examples Locally

All examples can be run locally using the same commands:

```bash
# Clone the repository
git clone https://github.com/your-org/wizardopus.git
cd wizardopus

# Install dependencies
pnpm install

# Run a specific example
cd examples/basic-form-wizard
pnpm dev
```

## Contributing Examples

We welcome contributions! To add a new example:

1. Create a new directory in `/examples`
2. Include a comprehensive README
3. Use modern patterns (`wizardWithContext`, `defineSteps`, `step`)
4. Add TypeScript typing throughout
5. Include tests where appropriate

See our [contribution guide](https://github.com/your-org/wizardopus/blob/main/CONTRIBUTING.md) for more details.
