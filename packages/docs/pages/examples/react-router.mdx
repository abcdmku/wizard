# React Router Wizard

## Path

Path: `examples/react-router-wizard`

## Highlights

Highlights:

- TanStack Router integration via `useTanStackWizardRouter`
- URL-to-wizard synchronization without method mutation
- component-per-step pattern with typed data

## Run

Run:

```bash
pnpm --filter react-router-wizard-example dev
pnpm --filter react-router-wizard-example build
```
