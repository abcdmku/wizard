# Advanced Branching

## Path

Path: `examples/advanced-branching`

## Highlights

Highlights:

- dynamic `next` decisions from role data
- `canEnter` access control per role
- typed context updates across branches
- no `as ReturnType<...>` or step-data casts

## Run

Run:

```bash
pnpm --filter advanced-branching dev
pnpm --filter advanced-branching build
```
